Capacitación – Módulo de Movimientos Bancarios

MDI y Acceso

Desde el MDI del sistema encontrarás el acceso al Módulo de Movimientos Bancarios dentro del menú Procesos. Este módulo permite registrar, consultar y administrar movimientos como depósitos, cheques, transferencias y ajustes.

Pantalla principal

La pantalla se divide en Encabezado (datos del movimiento) y Detalle (visualización de asientos y totales Debe/Haber). Los controles clave son:

  • Cuenta Origen: cuenta bancaria que registra el movimiento.
  • Tipo de Operación: depósito, retiro, cheque, transferencia, etc.
  • Conciliado: marcar Sí/No para conciliación bancaria.
  • Moneda y Tipo de Pago.
  • Cuenta Recibe: se habilita automáticamente para Transferencias.

Flujo para Nuevo movimiento

  1. Selecciona Cuenta Origen y Tipo de Operación.
  2. Ingresa Número de Documento, Fecha, Concepto y Beneficiario.
  3. Define el Monto y el estado de Conciliado.
  4. Si la operación es Transferencia, el sistema habilita Cuenta Recibe y su nombre asociado.
  5. Presiona Guardar; se generará el asiento contable automático (detalle) y se actualizarán los saldos.
El signo contable (+/-) se determina por el tipo de operación. El detalle registra el Debe/Haber correspondiente.

Edición de movimientos

Selecciona un registro del grid y pulsa Editar. Podrás modificar datos como documento, concepto, beneficiario, monto, moneda, tipo de pago y conciliado. Si cambias la Cuenta Origen o el Tipo de Operación, el sistema anula el movimiento original y crea uno nuevo con los datos actualizados (manteniendo la trazabilidad).

Anulación

Con Anular el sistema revierte automáticamente los saldos, marca el movimiento como ANULADO y conserva el historial. Los asientos afectados se ajustan acorde al signo de la operación.

Totales y validaciones

  • El grid muestra Débito, Crédito y la Diferencia calculada en tiempo real.
  • Se controlan estados (ACTIVO, ANULADO, PENDIENTE), montos positivos y campos obligatorios.
  • Los estilos del grid ayudan a identificar rápidamente el estado del movimiento.

Reportes

Desde este módulo puedes exportar o visualizar listados de movimientos filtrando por fechas, estado, cuenta y tipo de operación. Útil para conciliaciones y cierres.
